# Restaurant System MVP

Sistema backend para gestión de pedidos desarrollado con FastAPI.

## Descripción

Este proyecto permite gestionar pedidos de un restaurante mediante una API REST, incluyendo validación de datos, generación automática de IDs y control de estados.

## Funcionalidades

- Crear pedidos (POST)
- Listar pedidos (GET)
- Validación de datos con Pydantic
- ID automático por pedido
- Estado inicial de pedidos (pendiente)

## Tecnologías

- Python
- FastAPI
- Uvicorn
